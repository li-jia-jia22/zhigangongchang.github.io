业务架构

端-边-云-应用四层协同，打造完整智能安全监测体系

四层技术架构

端侧 · 数据采集层

Perception

STC51单片机作为低功耗核心控制器，通过OneWire总线驱动DS18B20温度传感器和DHT11温湿度传感器，实现高精度环境数据采集。温湿度数据同步显示于LCD1602液晶屏幕，便于现场实时查看。

STC51单片机 DS18B20温度传感器 DHT11湿度传感器 LCD1602显示

边缘侧 · 智能推理层

Edge AI

MaixCAM AI视觉模块搭载轻量化YOLOv11模型，实现本地化实时推理。支持安全帽佩戴检测、危险区域闯入识别、烟火实时监测等功能。离线运行能力确保断网情况下仍能持续监测和本地告警。

MaixCAM YOLOv11 离线大模型 0.6s响应

云端 · 数据管理层

Cloud

FastAPI高性能后端框架搭建云端服务，通过Node.js串口转发服务器接收下位机数据，实现温湿度与视觉告警数据的统一存储。MySQL数据库支撑历史记录查询、报表生成与数据分析。

FastAPI Node.js MySQL RESTful API

应用层 · 交互展示层

Application

Web管理后台提供多工厂统一管理、实时数据看板、历史记录查询、EDA数据分析等功能。微信小程序实现移动端便捷访问，支持实时监测、告警推送、远程控制，满足管理人员随时随地掌握安全动态的需求。

Web后台 微信小程序 实时推送 数据可视化
